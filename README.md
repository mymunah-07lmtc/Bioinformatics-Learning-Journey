# Bioinformatics Learning Journey

Welcome to my Bioinformatics Learning Journey!  
I am a Biochemistry-Microbiology student aspiring to study Biomedical Engineering. This repository documents my exploration of bioinformatics tools, databases, and concepts used in modern biological research.

## Goals
- Learn key bioinformatics resources  
- Understand biological data analysis  
- Explore computational approaches in healthcare  
- Build a foundation for future biomedical engineering studies

## Mini Projects
- **[Protein of the Week](Mini-Projects/Protein-of-the-Week.md)** – Weekly deep dive into a different protein using NCBI, UniProt, and PDB.
- **[Disease of the Month](Mini-Projects/Disease-of-the-Month.md)** – Exploring genetic diseases, their molecular basis, and bioinformatics resources.
- **[Biomedical Engineering Applications](Mini-Projects/Biomedical-Engineering-Applications.md)** – Connecting bioinformatics tools to real BME problems.
