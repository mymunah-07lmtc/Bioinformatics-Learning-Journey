# Reflections

## Challenges
The biggest challenge was understanding how biological data is organized in large databases. At first, I felt overwhelmed by the number of formats (FASTA, GenBank flat files, PDB files) and how to interpret the annotations. Another challenge was learning the jargon – E‑values, bit scores, coverage – when using BLAST. But after practicing a few searches, it started to make sense.

## Interesting Discoveries
I was surprised by the amount of biological information freely available online. I had no idea that anyone can download the entire genome of an organism or the 3D coordinates of thousands of proteins. Also, learning that BLAST can be used online without any programming was eye‑opening. It made me realize how accessible bioinformatics can be for a beginner.

## Future Goals
- Continue learning Python (especially libraries like Biopython)  
- Explore genomics – maybe start with bacterial genome analysis  
- Study computational biology algorithms (sequence alignment, phylogenetics)  
- Build my own bioinformatics projects, such as a small sequence search tool or a data visualisation of protein properties  
- Prepare for graduate studies in Biomedical Engineering where these skills will be applied to healthcare problems

This first week has built a solid foundation. I’m excited to keep learning!
